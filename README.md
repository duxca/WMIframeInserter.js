## How to use

### Browser

```js
<script src="lib/IframeInserter.js">
<script>
console.log( IframeInserter() );
</script>
```

### WebWorkers

```js
importScripts("lib/IframeInserter.js");

console.log( IframeInserter() );
```

### Node.js

```js
var IframeInserter = require("lib/IframeInserter.js");

console.log( IframeInserter() );
```
